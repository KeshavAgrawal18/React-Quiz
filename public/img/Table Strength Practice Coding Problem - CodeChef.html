Table Strength

You have NNN table legs, of different strengths.
Pillar iii can bear a weight of WiW_iWi​, and will break if it has to bear a larger weight.

You'd like to construct a table using some non-empty subset of these table legs.
When you place a weight on a table, its load is equally distributed to each of its legs.

For example, if you build a table with 444 legs, and place a weight of 181818 on it, each leg will need to bear a weight of 184=4.5\frac{18}{4} = 4.5418​=4.5.
So for instance, a table whose legs have strengths [4,4,5,6][4, 4, 5, 6][4,4,5,6] will not be able to bear this weight (the two legs with strength 444 will break), whereas a table with leg strengths [5,6,6,8][5, 6, 6, 8][5,6,6,8] will be able to bear it.

Find the maximum possible weight that a table built out of some of these NNN legs can bear, without any of the legs breaking.
It can be proved that this maximum weight is always an integer.

Note: Subsets need not be contiguous: for example, [1,3][1, 3][1,3] is a subset of [1,4,3,2][1, 4, 3, 2][1,4,3,2].

Input Format

  • The first line of input will contain a single integer TTT, denoting the number of test cases.
  • Each test case consists of two lines of input.
    • The first line of each test case contains a single integer NNN, the number of table legs you have.
    • The second line of each test case contains nnn space-separated integers W1,W2,…,WNW_1, W_2, \ldots, W_NW1​,W2​,…,WN​, denoting the weight each leg can withstand.

Output Format

  • For each test case, output on a new line the maximum weight the table can withstand by using some non-empty subset of table legs.

Constraints

  • 1≤T≤2⋅1041 \leq T \leq 2 \cdot 10^41≤T≤2⋅104
  • 1≤n≤2⋅1051 \leq n \leq 2 \cdot 10^51≤n≤2⋅105
  • 1≤w[i]≤1041 \leq w[i] \leq 10^41≤w[i]≤104
  • it is guaranteed that the sum of n over all test cases won't exceed 2⋅1052 \cdot 10^52⋅105.

Sample 1:

Input
Output

Explanation:

Test case 111: There are seven non-empty subsets of pillars. Let's look at each of them.

  • [1][1][1] - the maximum weight that can be held is 111.
  • [2][2][2] - the maximum weight that can be held is 222.
  • [3][3][3] - the maximum weight that can be held is 333.
  • [1,2][1, 2][1,2] - placing a weight of 333 or more is impossible because 32=1.5>1\frac{3}{2} = 1.5\gt 123​=1.5>1, but 222 is possible.
  • [1,3][1, 3][1,3] - placing a weight of 333 or more is impossible, but 222 is possible.
  • [2,3][2, 3][2,3] - placing a weight of 555 or more is impossible, but 444 is possible.
  • [1,2,3][1, 2, 3][1,2,3] - placing a weight of 444 or more is impossible, but 333 is possible.

So, it's best to choose two legs, namely 222 and 333: this gives us a table that can withstand a weight of 444.

Test case 222: Choose the single leg [10][10][10], and it can bear a weight of 101010.
This is the maximum possible.
